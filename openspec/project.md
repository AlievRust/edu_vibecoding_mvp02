# Проект: ai-triage-lab

## Назначение

Учебный MVP на FastAPI для первичной автоматической обработки обращений клиентов. Приложение классифицирует текст обращения, сохраняет результат в SQLite и поддерживает локальный запуск без внешней инфраструктуры.

## Цели

- Реализовать основной HTTP-эндпоинт `POST /triage`.
- Поддержать три режима обработки: `mock`, `broken`, `aliceai-llm`.
- Сохранять каждое обращение в локальную SQLite-базу.
- Ограничивать частоту запросов на уровне `client_id`.
- Обеспечить запуск локально и через Docker.

## Ограничения

- Это учебный MVP, а не production-сервис.
- Без авторизации, очередей, фоновых воркеров и веб-интерфейса.
- Хранилище только SQLite.
- Секреты и ключи только через переменные окружения.
- Пользовательская и техническая документация ведётся на русском языке.
